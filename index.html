What is Machine Learning?

Machine Learning (ML) is a subset of AI that allows systems to learn from data without being explicitly programmed.

ML models learn patterns from data and make predictions or decisions.

Types of Machine Learning

1. Supervised Learning

Uses labeled data to train models.

2. Unsupervised Learning

Finds hidden patterns in unlabeled data.

3. Reinforcement Learning

Models learn through rewards & penalties.

Python ML Example

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

data = pd.read_csv('data.csv')

X = data[['X_feature']]
y = data['Y_target']

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

model = LinearRegression()
model.fit(X_train, y_train)

new_value = pd.DataFrame({'X_feature': [15]})
prediction = model.predict(new_value)

print(prediction)
